# xfg-ddd-gateway 模块协作说明

## 模块定位
- 本模块是网关应用服务，负责对外提供 HTTP 入口与基础路由能力。
- 聚焦请求接入、参数校验、协议适配与下游服务调用编排。

## 变更边界
- 允许修改：启动配置、Controller 入口、网关侧请求转换与返回封装。
- 禁止修改：核心领域规则实现、与当前需求无关的跨模块业务逻辑。

## 依赖约束
- 依赖统一继承 `gateway-parent` 与平台 BOM，避免模块内分散定义版本。
- 通过标准 API 契约与公共能力协作，避免直接耦合下游实现细节。

## 提交前检查
- 校验入口参数与返回语义一致，异常路径可观测可追踪。
- 评估远程调用次数，避免出现明显 N+1 或重复调用问题。
- 删除未使用的控制器代码、无效配置与过期注释。
